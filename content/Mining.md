# Mining

## General Formula


```
n = Bv / (M * (1 + (0.1 * Vu)))
```

| Symbol | Meaning                                      |
| ------ | -------------------------------------------- |
| `n`    | # of nodes required                          |
| `Bv`   | Belt capacity (item per minute)              |
| `M`    | Base mining speed (item per minute per node) |
| `Vu`   | Vein Utilization level                       |

> [!important] Ores and rare veins are gathered at a fixed speed regardless of the difficulty level.
