# Mining

## General Formulas

$$
n = \frac{V}{M\cdot\left(1+\frac{k}{10}\right)}
$$

| Symbol | Meaning                                   |
| ------ | ----------------------------------------- |
| $n$    | Number of nodes required                  |
| $V$    | Transport volume (item per minute) |
| $M$    | Mining speed (item per minute per node)   |
| $k$    | Vein Utilization level                    |
