# Pumping

## General Formula


```
n = Bv / (50 + (0.5 * Vu))
```

| Symbol | Meaning                                      |
| ------ | -------------------------------------------- |
| `n`    | # of pumps required                          |
| `Bv`   | Belt capacity (item per minute)              |
| `Vu`   | Vein Utilization level                       |
